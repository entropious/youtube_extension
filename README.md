# YouTube Player for VS Code 📺

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Marketplace](https://img.shields.io/visual-studio-marketplace/v/entro.youtube-panel.svg)](https://marketplace.visualstudio.com/items?itemName=entro.youtube-panel)

![YouTube Player Screenshot](media/description.png)

Experience YouTube like never before, right from your editor! Watch tutorials, listen to music, or follow live streams without ever breaking your focus. 🚀

## ✨ Features

- 🚀 **Built-in Sidebar Player**: Watch videos in a dedicated sidebar view or a full editor tab.
- 📺 **Large-Screen Mode**: One-click to "Open in Panel" to move the player from the sidebar to a main editor column.
- 🕒 **Smart History & Titles**: Automatically saves your recently watched videos (up to 50) and fetches their real titles for easy browsing.
- 🔄 **Continuous Play**: Enable "Autoplay" to automatically discover and play related videos when the current one ends—perfect for music and background focus.
- ⚡ **Auto-Resume**: Automatically reloads your last video when you restart VS Code.
- 🛠️ **Universal Link Support**: Supports regular YouTube URLs, YouTube Shorts, and `youtu.be` short links.
- ⌨️ **Standard Hotkeys**: Control your media with global shortcuts:
  - `cmd+alt+p` (mac) / `ctrl+alt+p` (win/linux): Play/Pause
  - `cmd+alt+o` (mac) / `ctrl+alt+o` (win/linux): Next Video
- 🎨 **Modern Glassmorphism UI**: 
  - **Hover-to-Reveal**: Controls stay hidden until you need them, giving you more screen space.
  - **Sleek Interface**: Translucent, modern design that integrates perfectly with your VS Code theme.

## 🚀 Getting Started

1.  **Install** the extension from the VS Code Marketplace.
2.  **Open** the YouTube view from the Panel (sidebar).
3.  **Paste** a YouTube link into the input field and hit **Go**.
4.  **Enjoy** your content while you code!

## ⌨️ Commands

| Command | Description |
| --- | --- |
| `YouTube: Load URL` | Prompt for a YouTube URL to play. |
| `YouTube: Play/Pause` | Toggle playback state. |
| `YouTube: Next Video` | Skip to the next related video. |

## 🛠️ Configuration

The extension uses VS Code's global state to securely store your history and settings locally.
