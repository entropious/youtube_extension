# YouTube Panel for VS Code 📺

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Visual Studio Marketplace](https://img.shields.io/visual-studio-marketplace/v/alexey.youtube-panel.svg)](https://marketplace.visualstudio.com/items?itemName=alexey.youtube-panel)

Enhance your development workflow with the ability to watch YouTube directly within your editor. Whether it's a tutorial, some background music, or a live stream, **YouTube Panel** brings it all into your VS Code workspace.

## ✨ Features

- 🚀 **Built-in Sidebar Player**: Watch videos in a dedicated view without leaving your code.
- 🕒 **Smart History**: Remembers your recently watched videos (up to 10) with their titles.
- ⚡ **Seamless Integration**: Supports regular YouTube URLs, YouTube Shorts, and `youtu.be` short links.
- 🔄 **Auto-Resume**: Automatically reloads your last video when you restart VS Code.
- 🛠️ **Command Support**: Quickly load any video by pasting its URL via the `YouTube: Load URL` command in the Command Palette.
- 🌐 **External Open**: One-click button to open the current video in your default web browser.

## 🚀 Getting Started

1.  **Install** the extension from the VS Code Marketplace.
2.  **Open** the YouTube view from the Panel (usually at the bottom or sidebar).
3.  **Paste** a YouTube link into the input field and hit **Go**.
4.  Enjoy your content while you code!

## ⌨️ Commands

| Command | Description |
| --- | --- |
| `YouTube: Load URL` | Prompt for a YouTube URL to play in the panel. |

## 🛠️ Configuration

The extension uses VS Code's global state to securely store your history locally, keeping your workspace clean and synchronized.
